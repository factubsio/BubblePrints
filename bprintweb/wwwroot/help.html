BubblePrints Help

Searching

Type in the big search bar at the top of the window. BubblePrints will search all blueprints by name and do a fuzzy match, then return the matches in the "best" order. It doesn't get it right 100% of the time, but what you're looking for should be in the top few results.

Pressing ctrl-p will focus the search bar.

You can choose which blueprint to open by using the arrow keys and enter or by clicking on the name of the result

If your search begins with !bp_ the rest of the search string will be treated as a blueprint guid.

You can also search by type/guid/namespace:

  • t:Type
  • g:Guid (g: Guid will ignore a leading !bp_)
  • n:Namespace

For example, glitter t:Buff will fuzzy match by name on glitter, and restrict the results to only blueprints whose type matches Buff.

?type will show limit results to only blueprints that contain a sub-component with that type, for example: ?AreaEffect t:BlueprintAbility.

#this is a string will perform a full-text search through every blueprint, this will also search the (English) localised text.

Expanding & Collapsing

Clicking on the expand/collapse button will toggle the row and its children’s visibility. Double click on a row to toggle its (and its children’s) collapse state. Hold Control while toggling to toggle all the children of the node (instead of the node itself).

Following Links

To follow links (denoted by the exclamation point prefixing them), click the link.
